Constack keeps track of your conversation topics as a stack. Type to push elements; click the stack to pop. All data is stored locally, so only you can see it. All topics are erased entirely when you reload the page.


Put your cursor in the text area and press:

  • Return (with text entered) to push
  • Delete (with no text entered) to pop
